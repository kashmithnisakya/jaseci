- **Eject no longer exposes webhook, websocket, or scheduled targets as plain HTTP endpoints**: `jac build --as source` mounted every discovered walker and function as a `POST /walker/...` or `POST /function/...` route, including `@restspec(protocol=webhook|websocket)` targets and `@schedule(...)` background jobs that `jac serve` deliberately keeps off the HTTP surface. The ejected app now applies the same filter as `serve.endpoints`, so those targets are excluded from the generated routes and from `/__health`, and only the plain-HTTP walkers/functions are mounted.
