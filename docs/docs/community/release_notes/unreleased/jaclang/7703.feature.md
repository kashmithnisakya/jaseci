- **jac-scale: authenticated GitHub release lookups**: set `GITHUB_TOKEN` (or `GH_TOKEN`) on the deploy driver and `jac start --scale` release lookups authenticate against the GitHub API instead of hitting the anonymous 60-requests/hour rate limit. The token is sent only to `api.github.com` over https, attached as an unredirected header - never to asset downloads, redirect targets, or the Docker Hub probe.
