- **Feature: Format cache for `jac fmt`**: Add `jac fmt --cache` (`-C`) flag that skips files already known to be formatted, keyed on content digest + effective `[format]`/`[check]` config (including suppress settings) + formatter-pipeline fingerprint (+ logical path under `--lintfix`). Entries live under `.jac/cache/fmt-v1/` (including outside a project) with atomic writes and self-healing corruption recovery. Enabled automatically in `jac precommit`; `--staged --verify` preserves project logical paths while caching staged blob bytes. Caching is skipped with `--to_screen`. Annex failures are never cached as clean. The store is self-bounding: cache hits refresh entry recency, and writes opportunistically prune (at most once per day) entries unused for 30 days, evict oldest-first past 20k entries, and sweep temp files orphaned by crashed writers. No behavior change when `--cache` is omitted.
