- **Fix: `{#* *#}` Block Comments in JSX Body**: Comment-only brace expressions (`{#* comment *#}`) inside JSX element bodies are now a first-class syntactic construct that parses, round-trips through `jac format`, and compiles to nothing at runtime. Previously they caused a parse error. Bare `{}` and whitespace-only `{   }` remain parse errors, matching TSX behavior.
