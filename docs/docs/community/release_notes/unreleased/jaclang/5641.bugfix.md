- **Parser: Arrow Function Syntax Detected Early with Actionable Hint**: Using JavaScript arrow function syntax (`=>`) in Jac now emits error `E0025` with a `help:` hint showing the correct Jac lambda syntax. Previously this produced confusing cascade errors (e.g. "Missing `}`", "Missing `>`") with no guidance. A spurious `W2001` "may be undefined" warning for `=>` when running with `type_check=True` has also been suppressed.
