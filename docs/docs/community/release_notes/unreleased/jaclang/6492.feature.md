- **Feature: `jac eject` produces a standalone, zero-jaclang app**: `jac eject` now compiles a Jac project into a project-internal `ejected/` folder split into a FastAPI `backend/` (walkers and functions compiled to Python that import a vendored `_osp.py` runtime instead of jaclang, serving the `@jac/runtime` HTTP contract and the built frontend) and a `frontend/` (the `.cl.jac` UI compiled to JavaScript with a Vite setup). The result runs with only `fastapi` + `uvicorn` and the project's npm deps; no Jac toolchain or `jaclang` install is required. Re-running regenerates the folder without `--force` (it carries a `.jac-ejected` marker), and the output directory is overridable via `[eject].output` in jac.toml.
