- **Add: User identity management, password reset, and pluggable emailer**: New `POST /user/add-identity` and `POST /user/verify-identity` endpoints let an authenticated user attach additional identities (e.g. an email) and confirm them via a tokenized link. New `POST /user/forgot-password` and `POST /user/reset-password` endpoints implement a one-time-token reset flow (the forgot endpoint always returns 200 to avoid leaking account existence). Email delivery goes through a new `Emailer` abstraction with a built-in SMTP backend; users can plug in their own provider (e.g. SendGrid) by setting `[plugins.scale.emailer] provider = "myapp.email:CustomEmailer"` in `jac.toml`, and the dotted path is resolved via `importlib` at startup with no jac-scale code changes. Reset and verification tokens are SHA256-hashed at rest, single-use, and TTL-bounded; persisted in MongoDB when configured (with a TTL index) or in-memory otherwise.
