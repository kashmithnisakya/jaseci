- **WebSocket Overhaul**: Production-ready WebSocket stack with JWT auth handshake, heartbeat/idle close, per-target and per-user connection caps (plus a tighter `max_anonymous_per_target` cap for `pub` endpoints), message-size limits, and structured reject frames. Functions can now opt into `@restspec(protocol=APIProtocol.WEBSOCKET)` alongside walkers.
- **Pluggable WS Backplane**: Multi-worker broadcast fan-out via Redis pub/sub. Set `[plugins.scale.websocket].backplane = 'memory' | 'redis'` to choose explicitly, or omit it to auto-detect from `redis_url` (WS section first, then `[plugins.scale.database]` as fallback).
- **WebSocket Metrics & Config**: Prometheus counters for connections and broadcasts at `/metrics`, plus a new `[plugins.scale.websocket]` config block for backplane selection, message size, connection caps, and rate limits.
