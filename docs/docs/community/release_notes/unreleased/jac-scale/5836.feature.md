- **Feat: Pub/Sub MessageBroker with Redis Streams**: Introduces a `MessageBroker` abstraction for jac-scale with publish / subscribe / consume / ack semantics, configurable retry with dead-letter queue, and a Redis Streams broker. Off by default; enable via `[plugins.scale.pubsub]` in `jac.toml` and select a broker (`redis_streams` or `noop`).
