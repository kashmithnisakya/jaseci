- **Feat: Event-streaming broker**: Adds an `EventStreamBroker` abstraction (`jac_scale.events.broker`) with `publish` / `@subscribe` / `consume` / `ack`, retry with DLQ, and replayable offsets via `start_from`. Ships with `LocalBroker` (in-memory) and `RedisBroker` (Redis Streams); selection is automatic based on whether a Redis URL resolves. Off by default; enable via `[plugins.scale.events]` in `jac.toml`.
