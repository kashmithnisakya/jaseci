- **Feat: Event-streaming broker with LocalBroker and RedisBroker**: Introduces an `EventStreamBroker` abstraction (`jac_scale.events.broker`) with `publish` / `@subscribe` / `consume` / `ack`, configurable retry with dead-letter topic, replayable offsets via `start_from` (`"latest"` / `"earliest"` / a broker-specific position token), and at-least-once delivery. Two implementations ship in-tree: `LocalBroker` (in-memory, single-pod, zero deps; right for dev, tests, and small deployments) and `RedisBroker` (Redis Streams, durable, cross-pod). Selection happens automatically based on whether a Redis URL resolves: `[plugins.scale.events].url` first, then `[plugins.scale.database].redis_url`, falling back to `LocalBroker` when neither is set or the `redis` extra is missing. Off by default; enable via `[plugins.scale.events]` in `jac.toml`.
